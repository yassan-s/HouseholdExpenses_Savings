# 家計簿機能＋貯金アプリ
家計簿管理をしつつ、貯金状況を確認できるアプリ
<br>
<br>

## 目的：努力量を可視化し応援したい
「努力を可視化＋自己管理できるアプリ」を作りたいと思い、<br>
大切な「お金」を取り扱う家計簿と貯金を題材としました。
<br>
<br>

## 機能一覧
***
* 家計簿機能
* 貯金目標登録機能
* 貯金記録機能
* フォロー機能
* ログイン機能
***
<br>
<br>
